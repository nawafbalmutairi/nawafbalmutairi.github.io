Game Concept
Astro Dash — a 2D side-scrolling arcade game
Epic
For gamers who want a simple, engaging game, our game is a simple 2D arcade game with only one input that is relaxing and enjoyable and creates competition to achieve the highest score possible.

The player is an astronaut in space trying to reach the moon. Tapping the screen activates the jetpack to fly upward. The world scrolls infinitely, spawning aliens, lasers, coins, and powerups at random intervals. Difficulty increases with distance. Score is tracked by survival time and coins collected. Players can buy character skins, compete on leaderboards, and add friends.

— Design System & Decisions
UI Design Language
Consistent visual system across all 6 screens
Design System
Art Style
Pixel art (8-bit) — retro aesthetic inspired by classic arcade games. All sprites, backgrounds, and UI panels use a consistent pixel grid. Nytrock Global Font for the blocky, game-appropriate typography.
Layout
Unity Canvas system with anchored UI elements. Centred panels with semi-transparent red overlay on pixel-art background. Consistent Back button top-left. Bottom navigation bar on relevant screens.
Navigation
Hub-and-spoke model: Main Menu is the hub. Each feature (Shop, Leaderboard, Social) is a spoke with a Back button returning to the menu. Add Friend and Change Username are sub-screens of Social.
Colour System
Sky blue background with green pixel landscape. Red/pink semi-transparent panels for content areas. White input fields and buttons for contrast. Gold coins for currency. Green/red for online/offline status.
Interaction
Single-tap input model: Large, clearly labelled buttons. Minimal text input (username only). Optimised for mobile touch — all tap targets are generously sized for the target age group (18-25).
— User Stories & Features
Story 1
"As a gamer, I want a simple input scheme to make the experience easy to get started with"
Implement single-input detection
Create gameplay action triggered by single input
Add visual feedback for input
Story 2
"As a competitive player, I want to compare my score to others on a leaderboard"
Implement local score storage
Create UI to view top scores
Submit score to leaderboard when run ends
Story 3
"As an inexperienced gamer, I want low difficulty that increases as I learn"
Difficulty scaling based on time survived
Increase speed over time
Safety period for first 10 seconds
Story 4
"As a creative player, I want to customise my profile to compare with friends"
Change username in friend tab
Customise character styles from menu
Story 5
"As a collector, I want to collect and unlock new character styles"
Create skin definitions
Implement skin selection menu (Shop)
Story 6
"As a young gamer, I want to be the best by unlocking powerups"
Include powerup spawning
Add powerup types (slow time, double score)
Pickup detection and temporary effects
Story 7
"As a player, I want a unique challenge each time so I don't get bored"
Random obstacle spawning
Random coin placement
Random powerup intervals
Story 8
"As a player, I want to track multiple score types to encourage trying new techniques"
Create score categories (distance, coins, time)
Display multiple scores in-game
End-of-run summary with all score types
