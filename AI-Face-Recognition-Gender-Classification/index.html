AI Face Recognition
& Gender Classification

Building and evaluating DenseNet and ResNet convolutional neural networks on Microsoft Azure Machine Learning to classify facial images by gender — trained on Kaggle data and tested with real-time inference on a custom dataset.

Nawaf Almutairi
BSc Computer Science
Northumbria University
Platform: Microsoft Azure ML Designer
Aim

To build machine learning models in Azure to recognise faces of human beings and classify them into male and female genders. Two CNN architectures — DenseNet and ResNet — are trained, evaluated on a test set, and then tested via real-time inference on a separately collected custom dataset.

— Model Architectures: DenseNet vs ResNet
Architecture A
DenseNet
A convolutional neural network that uses dense connections between layers through Dense Blocks. Each layer receives inputs from all previous layers and passes its feature maps to all subsequent layers — promoting feature reuse, reducing parameters, and strengthening gradient flow.
Huang, G. et al. (2018) Densely Connected Convolutional Networks, arXiv.org.
VS
Architecture B
ResNet
A deep learning CNN that uses residual (skip) connections allowing gradients to flow directly through the network. This solves the vanishing gradient problem that prevented earlier architectures from scaling to hundreds or thousands of convolutional layers.
Datagen (2023) ResNet: The basics and 3 ResNet extensions.
